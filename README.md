# 🚀 React Project with Props and State

Welcome to this project developed with **React + Vite**, designed to help you practice fundamental ReactJS concepts through an interactive and dynamic interface.

This project was created as an academic activity to reinforce the use of:

- Reusable components
- Props
- State
- Hooks
- Events in React

---

# ✨ Overview

The application allows you to:

- Display reusable components  
- Render text and images dynamically  
- Manipulate state with `useState`  
- Capture events with `onClick` and `onChange`  
- Interact with a dynamic counter  
- View changes in real time
